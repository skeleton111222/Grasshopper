# Grasshopper simulation 
This project visualizes/ simulates **Grasshopper hopping movement** using **Scratch**.
For better performance, the project is intended to be run using **TurboWarp**.

---

6. **Enable Turbo Mode**

7. **Run the project**
   - Click the **green flag (when clicked in code one)**
